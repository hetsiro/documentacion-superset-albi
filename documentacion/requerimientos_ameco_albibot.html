Indicadores de Actividad

KPIs principales — pestana "Indicadores de Actividad"

Que pide el cliente
1.1
Dos pestanas: Indicadores de Actividad · Indicadores de Personas.
Este modulo cubre la primera pestana.
1.2
5 tarjetas KPI (Big Number):
— OTs creadas: total de OTs en el rango de fechas
— OTs terminadas: OTs con estado Resuelta o Cerrada
— Tasa de finalizacion: OTs terminadas / OTs creadas x 100
— Operaciones resueltas: operaciones con estado Resuelta
— OTs Backlog: OTs con tipo de actividad Backlog
1.3
Tendencia de OTs: Line Chart con dos series (OTs nuevas y OTs terminadas) agrupadas por dia.
Estado de OTs a la fecha fin: Bar Chart horizontal con la cantidad de OTs por estado (Abierta, Asignada, Detenida, Resuelta, Cancelada).
⚡
Filtro de fecha: el usuario gestiona el rango por OrdenTrabajoFechaInicio (OTs) y OperacionFechaInicio (Operaciones).

Indicadores de Personas

Pestana "Indicadores de Personas" — KPIs de tiempos, cumplimiento y personal

Que pide el cliente
2.1
5 tarjetas KPI:
— Tiempos asociados a la OT en HH: suma de tiempos declarados (DuracionReal + Traslado + Espera + Administrativos) de las operaciones de OTs resueltas, en horas
— Tiempo promedio de resolucion: suma de tiempos de ejecucion de las operaciones / OTs resueltas, en horas
— Tasa de cumplimiento: OTs resueltas dentro del plazo / OTs resueltas totales x 100
— Personal presente hoy: resolutores presentes (dia/noche)
— Supervisores presentes hoy: supervisores presentes (dia/noche)
Columnas y datasets necesarios
Vista OT — dataset: dbo.v_OrdenesTrabajo
Tiempos declarados (para HH)
OperacionTiempoDuracionReal OperacionResolucionTiempoTraslado OperacionResolucionTiempoEspera OperacionResolucionTiemposAdministrativos
Tiempo de ejecucion (para promedio)
OperacionResolucionTiempoEjecucion
SLA (para tasa cumplimiento)
OrdenTrabajoFechaFin OrdenTrabajoFechaFinReal
Estado y fecha
EstadoOrdenTrabajo OrdenTrabajoFechaInicio
Charts a configurar en Superset
🔢 Big Number — Tiempos asociados a la OT en HH
Titulo[Ameco] | Tiempos OT en HH
Datasetdbo.v_OrdenesTrabajo
Filtro fechaOrdenTrabajoFechaInicio (Time Range)
Filtro chartEstadoOrdenTrabajo IN ('Resuelta','Cerrada')
Sufijohrs
Metrica — Custom SQL
sql
ROUND(
    (SUM(ISNULL(OperacionTiempoDuracionReal, 0))
   + SUM(ISNULL(OperacionResolucionTiempoTraslado, 0))
   + SUM(ISNULL(OperacionResolucionTiempoEspera, 0))
   + SUM(ISNULL(OperacionResolucionTiemposAdministrativos, 0)))
  / 60.0, 1
)
🔢 Big Number — Tiempo promedio de resolucion
Titulo[Ameco] | Tiempo Prom. Resolucion
Datasetdbo.v_OrdenesTrabajo
Filtro fechaOrdenTrabajoFechaInicio (Time Range)
Filtro chartEstadoOrdenTrabajo IN ('Resuelta','Cerrada')
Sufijohrs
Metrica — Custom SQL
sql
ROUND(
    CAST(SUM(ISNULL(OperacionResolucionTiempoEjecucion, 0)) AS FLOAT)
  / NULLIF(COUNT(DISTINCT OrdenTrabajoId), 0)
  / 60.0, 2
)

SUM(ejecucion de todas las operaciones) / COUNT(DISTINCT OTs resueltas) / 60 → horas promedio por OT

🔢 Big Number — Tasa de cumplimiento
Titulo[Ameco] | Tasa de Cumplimiento
Datasetdbo.v_OrdenesTrabajo
Filtro fechaOrdenTrabajoFechaInicio (Time Range)
Filtro chartEstadoOrdenTrabajo IN ('Resuelta','Cerrada')
Sufijo%
Metrica — Custom SQL
sql
ROUND(
    CAST(COUNT(CASE WHEN OrdenTrabajoFechaFinReal <= OrdenTrabajoFechaFin THEN OrdenTrabajoId END) AS FLOAT)
  / NULLIF(COUNT(OrdenTrabajoId), 0) * 100, 1
)

OTs resueltas dentro del plazo (FechaFinReal <= FechaFin) / total OTs resueltas x 100

Pendientes / Preguntas al cliente
✅ Tiempos OT en HH: SUM(DuracionReal + Traslado + Espera + Administrativos) / 60 para OTs resueltas.
✅ Tiempo prom. resolucion: SUM(OperacionResolucionTiempoEjecucion) / COUNT(DISTINCT OTs resueltas) / 60.
✅ Tasa cumplimiento: OrdenTrabajoFechaFinReal <= OrdenTrabajoFechaFin → cumplio plazo.
✅ Personal presente / Supervisores: dataset AmecoAsistencia creado. Filtrar por AsistenciaTipo = 'E', AsistenciaFecha = hoy, Perfil = 'Supervisor' para supervisores.
✅ Tiempos reportados: tabla agrupada por OperacionResolutor con 5 columnas de tiempo en horas.
✅ Ranking carga: Bar Chart por OperacionResolutor, COUNT(OperacionId) ordenado DESC.
